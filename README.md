# 🎧 Walmart Earbuds Scraper  

## 📌 Overview  
Walmart Earbuds Scraper is a Scrapy-based web scraping project that extracts earbuds' details (prices, ratings, availability, etc.) from Walmart. It automates the data collection process for research, price comparison, and market analysis.  

## 🚀 Features  
- Scrapes earbuds' **name, price, rating, reviews, availability**, and more  
- Handles **pagination** to fetch multiple pages of results  
- Uses **Scrapy-Playwright** for dynamic content loading  
- Saves data in **JSON, CSV, or a database**  

## 🏗️ Installation  

### 1️⃣ Clone the Repository  

# Getting Started - Cheat Sheet

1. Clone this project: `git clone https://github.com//scrape-wallmart.git`
2. Create a Python Virtual Environment: `python venv venv`
3. Activate the Python Virtual Environment: `venv\Scripts\Activate`
4. Install Scrapy using pip: `pip install scrapy`
5. Listing the scrapy projects `scrapy list` 
6. Running the scrapy project: `scrapy crawl spidername`
